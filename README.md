# 動的ポータルサイト

バックエンドサーバーの状態に応じて、動的なリダイレクトを行うWebアプリケーション

## 実装例
portal: http://www.ashpile.net
local: http://portal.ashpile.net

## 使用方法

1. リポジトリをクローンします。
2. 必要な依存関係をインストールします。
    ```bash
    npm install
    ```
3. 開発サーバーを起動します。
    ```bash
    npm run dev
    ```
